# 📊 Project Overview

This project presents an **interactive Power BI dashboard** designed to analyze Airbnb booking and revenue performance across global markets.

The goal of this analysis is to transform raw booking data into meaningful insights that support **data-driven decision making**. The dashboard provides a clear view of **booking demand, revenue distribution, customer behavior, and cancellation patterns**.

Through this dashboard, stakeholders can easily identify **high-performing regions, profitable property types, and operational risks affecting revenue**.

---

# 🎯 Business Questions Addressed

This project answers several key business questions:

• Which regions generate the highest Airbnb revenue?  
• What property types receive the most bookings?  
• How significant are instant bookings in customer behavior?  
• How do cancellation patterns vary across countries and room types?  
• Which markets present opportunities for revenue growth?

# 🧩 Data Model

The dashboard is built using a **Star Schema Data Model** to improve analytical performance.

**Fact Table**

Bookings  
• Booking ID  
• Revenue  
• Cancellation Status  
• Instant Booking Indicator  

**Dimension Tables**

Listings  
• Property Type  
• Room Type  

Locations  
• Country  
• Region  

Guests  
• Guest Rating  

This structure allows efficient filtering, aggregation, and performance optimization in Power BI.

# 📈 Dashboard Pages

## 1️⃣ Booking Insights

This page analyzes **overall booking demand and customer behavior**.

Key Metrics:

• Total Bookings  
• Instant Bookings  
• Average Guest Rating  
• Host Ratings  

Visualizations include:

• Bookings by property type  
• Regional booking distribution  
• Global booking map  

**Insight:**  
Europe contributes the largest share of total bookings.

---

## 2️⃣ Cancellation Analysis

This page focuses on **booking reliability and cancellation trends**.

Visualizations include:

• Cancellation rate by country  
• Cancellation patterns by room type  
• Regional cancellation distribution  

**Insight:**  
Some room types show significantly higher cancellation rates.

---

## 3️⃣ Revenue Insights

This page evaluates **revenue distribution across regions and property types**.

Visualizations include:

• Revenue by country  
• Revenue by region  
• Revenue by property type  

**Insight:**  
Premium properties such as **penthouses and villas generate the highest revenue**.

---

# 📊 Key Insights

• Europe contributes the largest share of total bookings  
• Instant bookings represent a major portion of reservations  
• Premium property types generate higher revenue  
• Cancellation patterns vary across regions and room categories  

---

# 💡 Business Recommendations

• Promote **instant booking options** to increase conversions  
• Focus marketing efforts on **high-performing regions**  
• Encourage listing of **premium property types**  
• Optimize **cancellation policies** to reduce revenue loss  
• Use seasonal booking trends to guide pricing strategies
